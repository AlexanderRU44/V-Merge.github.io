# 🚀 V-Merge

**V-Merge** — минималистичный веб-инструмент для работы с конфигурационными ссылками (VLESS, VMess, Trojan, Shadowsocks). Он позволяет объединять списки из разных источников, удалять дубликаты и анализировать параметры серверов в удобном интерфейсе.

![Version](https://shields.io)

## ✨ Основные функции

*   **🔗 Объединение из URL:** Вставьте ссылки на Raw-файлы (например, из GitHub или Pastebin), и сервис соберет все данные воедино.
*   **🧹 Глубокая очистка:**
    *   Автоматическое удаление дубликатов строк.
    *   Удаление строк-комментариев (начинающихся с `#`).
    *   Очистка внутренних комментариев в ссылках.
*   **📊 Интерактивный анализ (Table View):** Автоматический парсинг протокола, адреса и порта для каждой ссылки в реальном времени.
*   **🎯 Умная навигация:** Нажмите на любую строку в таблице, чтобы мгновенно перейти к этой конфигурации во второй колонке.
*   **🔍 Поиск и подсветка:** Быстрый поиск по результатам с функцией циклической прокрутки к совпадениям.
*   **💾 Экспорт:** Быстрое копирование в буфер обмена или сохранение в формат `.txt`.
*   **🌐 Мультиязычность:** Полная поддержка русского и английского интерфейсов.

## 🛠 Технологический стек

*   **Frontend:** HTML5, CSS3 (Modern UI, GitHub Dark Theme).
*   **Logic:** Pure JavaScript (ES6+), Fetch API.
*   **Storage:** LocalStorage (для локального счетчика посещений).
